# CrackWifi
Strumento realizzato in Bash ideale per automatizzare attacchi WiFi (WPA/WPA2 - PSK) finalizzati all'ottenimento della password.
Strumento ideale per automatizzare attacchi WiFi (WPA/WPA2 - PSK) finalizzati all'ottenimento della password.
Lo strumento wifiCrack dispone di 2 modalità di attacco. La prima è l'attacco Handshake, dove in maniera automatizzata si gestisce tutto il necessario affinché, tramite un classico attacco di de-autenticazione e riconnessione da parte di una stazione, si ottenga un Handshake valido con cui poter successivamente lavorare per applicare la forza bruta.
La seconda modalità di attacco è il PKMID ClientLess Attack, che concentra la sua attenzione sulle reti wireless che non hanno client associati (Metodo Moderno).
